You can use a pointer to point to a function. This is used mainly to get object/class like behavior.

//Normal Function
int add(int a, int b){
    return a + b;
}

//Pointer Function
int (* add_pointer_function)(int a, int b) = add

//add(4,5) is the same as add_pointer_function(4,5)


To pass the pointer function, use typedef

typedef int (*compare_cb) (int a, int b);
